# 🚗 Car Recommender System

A machine learning-based recommendation system that suggests cars based on user preferences. The project combines web scraping, data preprocessing, and predictive modeling to generate personalized recommendations.

---

## Features

- Web scraping for collecting car information
- Data cleaning and preprocessing
- Feature engineering
- Machine learning model training
- Personalized car recommendations

---

## Technologies

- Python
- MySQL
- BeautifulSoup
- Pandas
- NumPy
- Scikit-learn

---

## Results

The trained recommendation model predicts suitable cars based on user preferences and similarity between vehicle features.
